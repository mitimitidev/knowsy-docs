Break-even volume

At $40/mo fixed costs and ~$9.21 net revenue per report (after Stripe), break-even is reached at approximately 5 reports / month. This is a very low bar for a consumer product.

WRO fallback cost scenario

If WRO is blocked — DataTrace API fallback

DataTrace charges approximately $2–4 per document retrieval. A 5-deed chain would add $10–20 in data costs per report, making the current $9.99 price point unprofitable. In this scenario, pricing would need to move to $29–49 or a hybrid retrieval model (WRO first, vendor fallback only on failure) would be required to maintain margins.

This is the primary cost risk and the key reason WRO automation is treated as a P0 technical requirement in Phase 3.
