Data sources & vendor evaluation

Assessment roll — address to owner lookup

Westchester County
Direct county assessment roll data. Free, updated annually. Accessed via county GIS portal.
Recommended
ATTOM Data
Aggregated property data API. Reliable, real-time, but adds per-call cost.
Fallback
Regrid
Parcel data coverage strong in NY. Good for normalized parcel IDs across counties.
Fallback

Deed retrieval — recorded documents

WRO (direct)
Westchester Records Online. Free public portal. Primary path — automated login and document download.
Primary / risk
DataTrace
Enterprise title data provider. Full Westchester coverage, API-based. High cost per pull.
Fallback
PropLogix
Title search support vendor. Can retrieve WRO documents on demand. Human-in-the-loop — slower.
Manual fallback

OCR & extraction workflow

1

PDF download

Deed image downloaded from WRO and stored in Vercel Blob. Original file preserved for audit trail.

2

OCR (Google Document AI)

Preferred OCR vendor. Handles variable scan quality, handwriting, and stamped text. Returns word-level confidence scores — used to flag low-confidence extractions for review.

3

AI deed parsing (Claude)

Structured extraction via tool-use: grantor, grantee, recorded date, consideration, legal description, encumbrances, prior deed book/page reference. Output is validated against a JSON schema before acceptance.

4

Prior-deed chaining

The extracted prior-deed reference (book and page number) feeds back into the retrieval step. Loop continues until no prior reference is found or the records pre-date available digital archives (typically pre-1960).

5

Report assembly

All parsed deed records written to Postgres. Report generation job marks complete. User receives email with report link.

Pipeline monitoring

Job state machine

  • pending → queued → retrieving → ocr → parsing → complete
  • Any step can transition to: failed, needs_review
  • Status exposed to frontend via polling endpoint

Failure modes tracked

  • WRO login failure / CAPTCHA block
  • Document not found in archive
  • OCR confidence below threshold (70%)
  • AI extraction schema validation failure
  • Prior-deed loop exceeded max depth (20)
